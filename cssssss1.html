What is CSS ?

CSS stand for Cascading Style Sheet, is used to set the style or presentation of document written in markup language. It provides easy alternatives to specify various attributes of tags.

Types of Style Sheets

  • Inline Style Sheet
  • Embedded/Internal Style Sheet
  • External Style Sheet

Inline Style Sheet

When we want to apply the styles on a particular HTML element, then this Style Sheet is being used. This overrides the style coded in Embedded or External Style Sheet.
Example : <.... style="color:blue;font-size:18px;">
