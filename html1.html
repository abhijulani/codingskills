What is HTML?

HTML stands for Hyper Text Markup Language, is the standard markup language which is used to create web pages. HTML is based on Standard Generalized Language (SGML). It is also used to create user interfaces for mobileand web applications. Web pages can be static or dynamic.

Advantages of HTML

  • User Friendly
  • It is flexible to use
  • Easy to use and understand
  • All browsers support HTML
  • HTML and XML syntax is very small
  • HTML is free
  • Easy maintenance and updation

Disadvantages of HTML

  • Security features offered by HTML are limited
  • Insufficent for Dynaic Pages
  • Limited for Displaying Content
  • Unpredictable Behavior across Browsers
